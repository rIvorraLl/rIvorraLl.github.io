Cthulhu web

A web project built from scratch for a fake publishing house.

Built using Vue and Bootstrap, with a fake API served by json-server. It works as a SPA (Single Page Application).

The project code can be found in my GitHub repository.

To test the code (in Linux/UNIX-like shell):

  • You'll need a web client (browser), npm and json-server installed.
  • Open a terminal emulator. Clone the repository in your local machine: git clone https://github.com/rIvorraLl/diw_proyecto_2t.git
  • Navigate inside the cloned directory: cd diw_proyecto_2t/
  • Install dependencies: npm install

  • Run json-server with the provided json file: json-server --watch data_db.json

    It should print the URLS for the endpoints, which you can check introducing the URL in your browser.

  • Run npm run dev. You may need to open a new terminal window for this, as json-server will have the former one busy. Now if you enter the URL in your browser (most likely http://localhost:5173/) you can now navigate the web.
  • You'll find a link labeled "Identificación" at the footer. You can login with the credentials "rafael" and "patata123". Change these credentials modifying the provided data_db.json file if you wish.
  • Once logged in, you can add, delete and modify data for the books. The changes will be reflected on the fly.
